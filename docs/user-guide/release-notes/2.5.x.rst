======================
Pulp 2.5 Release Notes
======================

Pulp 2.5.2
==========

New Features
------------

The generation of sqlite files is now done with createrepo_c instead of createrepo.

Pulp 2.5.0
==========

New Features
------------

The Alternate Content Source feature now works during an rpm sync.

---------
